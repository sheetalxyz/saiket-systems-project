# Saiket Systems Python Internship

## About
This repository contains the Python projects completed during my internship at Saiket Systems.

## Projects

### Task 1 - To-Do List
A Tkinter-based To-Do List application to add and delete daily tasks.

### Task 2 - Guess the Number Game
A Python GUI game where the user guesses a randomly generated number.

### Task 3 - File Handling Tool
A Tkinter application for reading and displaying text file content with error handling.

### Task 4 - Web Scraper
A Python GUI application that extracts webpage information using web scraping.

### Task 5 - Currency Converter
A simple Tkinter-based currency converter application.

### Task 6 - Word Count Tool
A GUI application that counts words, lines, characters, and displays word frequency.
